# Data directory

Raw Argoverse 2 data are not included in this repository.

To reproduce the large-scale report run, first convert AV2 scenarios into the generic trajectory CSV format expected by CausalSensor4D and place the selected clean-safe interaction CSV files in:

```text
data/clean_safe_interaction_csv/
```

Toy CSV examples are provided under `examples/batch_csv_scenes/`.
