# GitHub upload list

Upload the whole contents of this folder as one public repository.

Include:

- `src/`
- `examples/`
- `scripts/`
- `data/README.md`
- `README.md`
- `REPRODUCIBILITY.md`
- `requirements.txt`
- `pyproject.toml`
- `.gitignore`
- `LICENSE`
- `LICENSE_NOTES.md`

Do not upload:

- raw AV2 data;
- full `outputs/` directories with generated candidate tables;
- old version folders as separate copies;
- local IDE or runner files with hard-coded machine paths;
- `.venv`, `__pycache__`, `.idea`, `.vscode`;
- API keys, `.env` files, tokens, or server credentials;
- private figures, tables, drafts, reviewer files, or release materials.
